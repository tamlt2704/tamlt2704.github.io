    python functiontools provides bunch of usefull tools to work with functions. In this post I will introduce some application of this module to make our life easier.

    1. wrap function

    Example if I want a program to read a file that contains multiple numbers, print True or Fasle if it is a prime number or not

    It can be easily implementation as bellow:

    functool_wrap_ex1.py (Source)

    import sys

def is_prime_number(n):
        if any([n % i == 0 for i in range(2, n/2)]):
                return False
        return True

def find_prime_number(filename):
                with open(filename, 'r') as f:
                        for line in f:
                                n = int(line)
                                print n,is_prime_number(n)

if __name__ == "__main__":
        find_prime_number(sys.argv[1])

"""
INPUT           Output
        3       3 True
        10      10 False
        15      15 False
        17      17 True
"""

    If the file does not exists, the program will thrown an exception. We can avoid that by adding try catch as bellow

    find_prime_number(filename):
        try:
                with open(filename, 'r') as f:
                        for line in f:
                                n = int(line)
                                print n,is_prime_number(n)
        except Exception as e:
                print e
                pass
    • The code will print the exception instead of crashing now.
    • Imagine if we have to check exception for multiple functions in our code. It will not a good idea to have multiple try / catch. wraps is a nice built-in function to make it less frustruated.

    functool_wrap_ex2.py (Source)

    import sys
from functools import wraps
from datetime import datetime

def exception_wrapper(f):
        @wraps(f)
        def wrapper(*args, **kwargs):
                try:
                        f(args, **kwargs)
                except Exeption as e:
                        print e
                        pass
        return wrapper


def running_time(f):
        @wraps(f)
        def wrapper(*args, **kwargs):
                t1 = datetime.now()
                f(args, **kwargs)
                print 'function', f.__name__ , 'ran in ', datetime.now() - t1
        return wrapper

def is_prime_number(n):
        if any([n % i == 0 for i in range(2, n/2)]):
                return False
        return True

@running_time
@exception_wrapper
def find_prime_number(filename):
        try:
                with open(filename, 'r') as f:
                        for line in f:
                                n = int(line)
                                print n,is_prime_number(n)
        except:
                pass

if __name__ == "__main__":
        find_prime_number(sys.argv[1])

"""
INPUT           Output
        3       3 True
        10      10 False
        15      15 False
        17      17 True
"""
    • As you can see, we can even monitor the run time of the function without changing it internal code.

    Objective: To move N disks from column one (C1) to colum three (C3).

    1. First we have to find a way to move top N-1 disks from C1 to C2 using C3 as a temporary column.

    2. Now there are one disk left at C1, move it directly from C1 to C3

    3. Next Step is to move N-1 disks from C2 to C3 using C1 as temporary column

    hanoi_tower.py (Source)

    def move(nb_disk, origin, target, tmp):
        if nb_disk >= 1:
                move(nb_disk - 1, origin, tmp, target)
                print 'move disk from {} to {}'.format(origin, target)
                move(nb_disk - 1, tmp, target, origin)

move(3, 1, 3, 2)

    1. Why nikola

      I used to use jekyll for blogging but I always have troubles with installing ruby and their dependencies. Futhermore, modifying the blog in a language that I'm not familiar is not a good idea. Since I already have python in my system, adding nikola is just one line of code. Start with something I know gives me the power to overcome the procastination j.

    2. Start blogging with nikola and publish to github

      Create a repository on github follow the name convention <username>.github.io

      Clone repository

      git clone http://github.com/<username>/<username>.github.io

      Install nikola

      pip install nikola

      Init new blog

      nikola init .

      Create new post

      nikola new_post
<edit your post>

      Create new page

      nikola new_page

      View your blog

      nikola build && nikola serve

      Publish to github

      nikola github_deploy

    Enjoy your blog at http://<username>.github.io and happy blogging!
